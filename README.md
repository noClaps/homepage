# ZeroLimits.dev
This is the source code for [ZeroLimits.dev](https://zerolimits.dev).

## Build instructions
1.  ```bash
    git clone https://github.com/noClaps/homepage && cd homepage
    ```
2.  ```bash
    npm install
    npm run dev
    ```
    Run this in the root directory (`homepage/` if you followed the instructions above) and open `localhost:3000` in your desired browser to see the preview. Make sure to keep this repository and your dependencies up-to-date, especially if you plan to contribute.
3.  ```bash
    npm run build
    ```
    This builds the site and places the files in the `dist/` directory.
